# correlation_paper

The folder contain:

(1) Eight .dat files containing the time evolution of the final separation (one for each simulation),

(2) Eight .dat files containing the time evolution of the unbound mass fraction (one for each simulation),

(3) Two .dat files with the data presented in Tables~4 and 5,

(4) A Python script to generate Figure 7.

All figures can be easily reproduced using the files and script provided.

The filenames are self-explanatory. If you have questions, feel free to reach out.
